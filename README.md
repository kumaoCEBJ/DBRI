## 右クリックの特殊操作
* フィルター : 指定した要素を除外対象にする(種族/装備)
* スキル選択 : 指定したカテゴリを入力欄にコピー
* シミュ : 選択したスキルをOr対象にする⇔選択したスキルを除外対象にする

## ユニットの倍率計算に使用するスキル
* ｢多段攻撃/全力攻撃/心核穿ち/必殺増加/致命必殺/カブト割/次元斬撃｣
  * 確率追撃は未使用

## ゲーム内の説明と一致しなかったもの
  * 日中/夜戦適応 : マイナスの地形効果は防げない

## シミュ
* 撃破金運のみ重複指定可能
  * 重複指定と除外指定が重なった場合は例外的に除外を無効にする
